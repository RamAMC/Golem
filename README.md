## 📖 Descripción  
Peuqeña web app de edicion de imagenes para imprimir y utilizar en la construccion de folioscopios mecanicos.

---

## 🚀 Características  
- 📂 Carga y previsualización de imágenes  
- ✂️ Ajuste de relación de aspecto y tamaño 
- 📦 Descarga en ZIP
